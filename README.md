# Digital Couture Designer

A web application for designing and customizing outfits digitally.

## Features
- User login (phone-based)
- Create new designs
- Save measurements
- View design history

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot
- Database: MySQL

## Setup
1. Run backend:
   ./mvnw spring-boot:run
2. Open frontend login.html
